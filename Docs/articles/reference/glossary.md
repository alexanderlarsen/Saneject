# Glossary

## Injection target

`Component` with injected fields, properties or methods.

## Injection site

Injected field, property or method.

## Scope

`MonoBehaviour` that declares bindings for a part of your hierarchy.

## Binding

Instruction declared in a `Scope` that tells Saneject what to resolve, how to inject it - an optionally where to inject it.
